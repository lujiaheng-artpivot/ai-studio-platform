🎯 GitHub Desktop 操作指南

跟着这5步操作，3分钟完成部署！

1 打开GitHub Desktop并登录

打开刚安装的GitHub Desktop应用

如果是第一次使用：

  • 点击 Sign in to GitHub.com
  • 在浏览器中完成登录
  • 授权GitHub Desktop访问
  • 返回GitHub Desktop
💡 提示： 使用您的GitHub账号 lujiaheng-artpivot 登录
2 添加本地仓库

在GitHub Desktop中：

  1. 点击菜单栏 File → Add Local Repository...
  2. 点击 Choose... 按钮
  3. 导航到并选择这个文件夹：
    /Users/lujiaheng/Documents/ai-studio-platform
  4. 点击 Add Repository
⚠️ 重要： 确保选择的是 ai-studio-platform 文件夹，不是其他文件夹
3 发布到GitHub

仓库添加成功后，您会看到：

  1. 中间会显示一个蓝色按钮：Publish repository
  2. 点击这个按钮
  3. 在弹出的窗口中：
    • 确保 Name 是：ai-studio-platform
    • 确保 勾选了 Keep this code private 的选项是 未勾选（公开仓库）
    • 或者勾选 Public 选项
  4. 点击 Publish Repository 按钮
💡 提示： 发布过程需要几秒钟，请耐心等待进度条完成
4 确认发布成功

发布完成后：

  • GitHub Desktop顶部会显示 No local changes
  • 右键点击仓库名称，选择 View on GitHub
  • 浏览器会打开您的GitHub仓库页面

您的仓库地址应该是：

https://github.com/lujiaheng-artpivot/ai-studio-platform
5 部署到Vercel

代码已经在GitHub上了，现在部署到Vercel：

  1. 点击下方按钮打开Vercel
  2. 点击 Continue with GitHub 登录
  3. 点击 Import Project
  4. 找到并选择 ai-studio-platform 仓库
  5. 点击 Import
  6. 保持默认设置，点击 Deploy
  7. 等待2-3分钟，部署完成！🎉
🚀 打开Vercel部署页面
