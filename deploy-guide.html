🚀 一键部署到公网

3分钟让全世界看到您的AI Studio

cd /Users/lujiaheng/Documents/ai-studio-platform && bash one-click-deploy.sh

1打开终端

点击上方按钮会自动打开终端并复制命令

2粘贴并执行

在终端中按 Cmd+V 粘贴命令，然后按回车执行

3登录Vercel

浏览器会自动打开，选择 GitHub 或 Email 登录

4回答问题

终端会问几个问题，都可以直接按回车使用默认值

5等待部署

Vercel会自动构建和部署，大约需要2-3分钟

💡 部署提示

  • 登录时推荐选择 GitHub（最快）
  • 项目名称可以使用默认的 ai-studio-platform
  • 所有问题都可以直接按回车
  • 部署完成后会显示公网地址
  • 免费套餐完全够用，无需付费

🎉 部署完成后

您将获得一个公网地址，例如：
https://ai-studio-platform-xxx.vercel.app

全世界的人都可以访问您的网站！
