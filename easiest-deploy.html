🚀 一键部署命令

复制下面的命令，在终端中执行

cd /Users/lujiaheng/Documents/ai-studio-platform && bash super-simple-deploy.sh

📋 执行后会发生什么？

1 浏览器自动打开
Vercel登录页面会自动打开
2 登录Vercel
使用GitHub或Email登录
3 回答几个问题
在终端中按提示回答（都可以直接按回车）
4 自动部署
Vercel会自动上传和部署代码
5 获得公网地址
2-3分钟后，您的网站就上线了！
💡 回答问题时的建议：
  • Set up and deploy? → Y
  • Which scope? → 选择你的账号
  • Link to existing project? → N
  • Project name? → 直接按回车
  • In which directory? → 直接按回车
  • Override settings? → N
